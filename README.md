# Flora Visionaries - Plant Landing Page

## Overview

Flora Visionaries is a responsive plant landing page built using React.js and Tailwind CSS. The project recreates a modern plant e-commerce landing page with reusable components and responsive design.

## Tech Stack

* React.js
* Vite
* Tailwind CSS
* React Icons

## Features

* Responsive Navigation Bar
* Hero Section
* Trending Plants Section
* Top Selling Plants
* Customer Reviews
* O₂ Plant Section
* Footer
* Mobile Responsive Design
* Hover Animations

## Installation

Clone the repository:

```bash
git clone https://github.com/Kamalakar30/flora-visionaries.git
```

Go to the project folder:

```bash
cd flora-visionaries
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```
